# Review: Sales

## Preconditions
- Read `PRD.yml`, `roadmap.yml`, `_contract.md`
- Identify environment (local/staging/production)

## Review Areas

### Value Proposition
- Hero headline clarity (5-second test)
- USP differentiation
- Benefits vs features balance
- Score: 1-10

### CTAs
- Inventory all CTAs (primary, secondary, micro)
- Action-oriented language
- Visual prominence and placement
- Test each conversion flow

### Lead Capture
- Form field count (friction)
- Validation and error handling
- Newsletter value proposition

### Trust Signals
- Client logos, testimonials
- Case studies, certifications
- Proximity to conversion points

### Sales Funnel
- Map: Awareness → Interest → Consideration → Intent → Conversion
- Identify drop-off points
- Industry-specific journeys

## Output
- Conversion readiness score (1-100)
- Top 5 priorities
- Quick wins (<1 day)
- Record issues in `.seedgpt/bugs/` or `.seedgpt/minor-improvements/`
- Commit updates to `main`

See: `_contract.md`
